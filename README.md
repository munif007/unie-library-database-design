# UNIE Library Database Design

This project demonstrates the design and implementation of a university library database using Microsoft SQL Server and T-SQL.

The database was developed through conceptual, logical, and physical database design stages. It includes an EER model, relational schema design, normalization, SQL Server table creation, constraints, triggers, sample data, and reporting queries.

## Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio
- T-SQL
- Enhanced Entity Relationship (EER) modelling
- Relational database design
